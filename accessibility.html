Accessibility Statement

This accessibility statement applies to the StudyNest website.

We want as many people as possible to be able to use this website. For example, that means you should be able to:

  • Zoom in up to 300% without the text spilling off the screen.
  • Navigate most of the website using just a keyboard.
  • Listen to most of the website using a screen reader.

Compliance Status

This website is partially compliant with the Web Content Accessibility Guidelines version 2.2 AA standard.

Non-accessible Content

We know some parts of this website are not fully accessible:

  • Some PDF past papers are scans and may not be readable by screen reader software.
  • The RAG (Red/Amber/Green) tracker relies on color to convey meaning. We are working on adding text labels to these buttons.

Feedback and Contact Information

If you need information on this website in a different format like accessible PDF, large print, or easy read, or if you find any problems not listed on this page, please contact us:

Email: support@studynest.com

We will consider your request and get back to you within 7 days.

Enforcement Procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018. If you are not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).
